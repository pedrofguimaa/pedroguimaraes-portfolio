# 🚀 Portfólio Front-End — Pedro Guimarães

Portfólio pessoal desenvolvido com foco em interfaces modernas, experiência do usuário e organização visual, apresentando projetos, habilidades e abordagem em desenvolvimento front-end.

---

## ✨ Sobre o projeto

Este portfólio foi criado com o objetivo de demonstrar minhas habilidades em desenvolvimento front-end, com foco em:

- construção de interfaces com qualidade visual
- organização e estrutura de código
- experiência do usuário
- identidade visual consistente
- abordagem orientada a produto

O projeto possui uma estética baseada em sistema/interface digital, trazendo uma experiência mais imersiva e diferenciada.

---

## 🧠 Destaque: Rift Codex

Projeto principal do portfólio.

Uma interface autoral inspirada no universo de **League of Legends**, focada em organizar informações de Runeterra em uma experiência interativa.

### 🔍 Características:

- navegação por regiões
- visual imersivo
- organização de dados
- design inspirado em produto real
- interface responsiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git & GitHub

---

## 🎨 Funcionalidades

- layout moderno e responsivo
- sidebar com links profissionais
- cursor customizado
- páginas de case detalhadas para projetos
- sistema de navegação entre páginas
- organização modular de código (css/js separados)
